# Vyber v0.1.0 — First Public Release

The first official release of Vyber, a soundboard that plays audio through your speakers and into voice chat simultaneously using VB-CABLE.

## Features

- **Soundboard with virtual mic routing** — play sounds through speakers and Discord/voice chat at the same time
- **Category tabs** — organize sounds into custom categories with drag-and-drop reordering
- **Global hotkeys** — trigger sounds from any application
- **Per-sound volume control** — adjust individual sounds from 0% to 200% for quiet clips
- **Output modes** — Speaker Only, Mic Only, or Both
- **Mic Passthrough** — mix your real microphone audio so friends hear both you and your sounds
- **VB-CABLE integration** — built-in installer and setup guide
- **Discord Setup guide** — step-by-step instructions for configuring Discord
- **System tray** — minimize to tray, restore on click
- **Bug reporter** — submit issues directly to GitHub from within the app
- **Auto-updater** — manual update check from the system tray

## Notes

- Requires Windows 10 or later
- VB-CABLE is required for mic routing (Vyber will prompt you to install it)
- Supports WAV, MP3, FLAC, and OGG audio formats
