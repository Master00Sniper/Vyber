# Vyber v0.1.4

## Changes

- **Manual update flow** — Update check now directs users to vyber.mortonapps.com instead of auto-downloading, avoiding Windows Defender blocking the swapped exe
- **Discord Setup redesign** — New card-based step layout with numbered badges, highlighted settings, and styled toggle indicators for a cleaner look
- **Discord Setup button** — Moved from the hamburger menu to a prominent button on the main screen for easier access
- **Website SEO overhaul** — Added Open Graph / Twitter Card meta tags, JSON-LD structured data, sitemap, canonical URLs, and improved heading hierarchy
- **Mobile hamburger menu** — Website navigation collapses into a hamburger menu on mobile
- **Website polish** — Darkened hero background, adjusted text positioning, added Morton Apps logo to callout section
- **Credits section** — Added Icons8 attribution to the About dialog
