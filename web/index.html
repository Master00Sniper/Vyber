Features

Dual Audio Output

Play sounds through your speakers and virtual microphone simultaneously. People in voice chat hear your sounds instantly.

Zero Configuration

Install VB-CABLE once and the app handles all the audio routing automatically. No manual setup needed.

Mic Passthrough

Your voice and soundboard audio are mixed together so you can talk and play sounds at the same time.

Global Hotkeys

Trigger sounds with keyboard shortcuts that work even while you're in a game or another app.

Organized Library

Organize your sounds into tabbed categories. Add, rename, and manage sounds with an intuitive interface.

Instant Stop

Cut all audio immediately with a stop-all button or hotkey. Stay in control at all times.

Getting Started

1

Install VB-CABLE

Download and install VB-CABLE (free). Reboot your PC.

2

Install Dependencies

Run pip install -r requirements.txt in the project directory.

3

Launch & Play

Run python run.py, add your sounds, and set your voice chat input to CABLE Output.
